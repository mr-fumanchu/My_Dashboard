Gmail / Google Account

Quickest setup: Generate an App Password at myaccount.google.com/apppasswords (requires 2-Step Verification). Use it below — Gmail's IMAP/SMTP settings will be filled in automatically. Your account is stored locally in your dashboard.

IMAP: imap.gmail.com:993 SSL  |  SMTP: smtp.gmail.com:587 TLS
Proper OAuth 2.0 setup — 5 steps:
1. Go to Google Cloud Console → Create a project → Enable the Gmail API.
2. Go to APIs & Services → OAuth consent screen → configure it.
3. Go to APIs & Services → Credentials → Create OAuth 2.0 Client ID (type: Web application).
4. Add this exact redirect URI: http://localhost:3000/oauth/google/callback
5. Paste the Client ID and Client Secret below, click Save Credentials, then click Connect Google Account.
